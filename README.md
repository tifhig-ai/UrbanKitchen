# Urban Kitchen

Static flyer assets for Urban Kitchen.

## Run locally

From this folder:

```sh
python3 -m http.server 8000
```

Then open:

- http://localhost:8000/urban-kitchen-four-up-flyer.html
- http://localhost:8000/urban-kitchen-flyer-burger-alt.html
- http://localhost:8000/urban-kitchen-flyer-final-qr-fix.html
